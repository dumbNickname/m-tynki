# M-TYNK Wrocław - Tynki ze Szlichtą pod Malowanie

Profesjonalna strona internetowa dla firmy M-TYNK specjalizującej się w tynkach gipsowych ze szlichtą gładziolitu wygładzającego.

## Technologie

- **SolidStart** - Framework do budowy aplikacji z SSG
- **Solid.js** - Reaktywna biblioteka UI
- **Tailwind CSS** - Framework CSS
- **TypeScript** - Typowanie statyczne

## Funkcje

- ✅ Statyczne generowanie stron (SSG)
- ✅ Pełna responsywność
- ✅ Optymalizacja SEO
- ✅ Structured Data (Schema.org)
- ✅ Open Graph i Twitter Cards
- ✅ Dostępność (a11y)
- ✅ Szybkie ładowanie
- ✅ Nowoczesny design
